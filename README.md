# AI-ML-Internship-Task-12
## Task 12: KMeans Customer Segmentation

### Objective
Perform customer segmentation using KMeans clustering.

### Dataset
Mall Customer Segmentation Dataset (Kaggle)

### Tools Used:
- Python
- Scikit-learn
- Matplotlib / Seaborn


### Steps
1. Loaded dataset and selected income & spending score.
2. Applied StandardScaler.
3. Used Elbow Method to find optimal clusters.
4. Trained KMeans model.
5. Visualized clusters.
6. Saved segmented dataset.


  ### Final Outcome

- Performed KMeans clustering on mall customer data.
- Used Elbow method to find optimal K value.
- Visualized customer segments.
- Exported segmented dataset.
- Interpreted business meaning of clusters.

Intern can now perform unsupervised customer segmentation for business use.
